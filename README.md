### How to Use Reusable Prompts

**To Install:**
- Click the **VS Code** or **VS Code Insiders** install button for the prompt you want to use
- Download the `*.prompt.md` file and manually add it to your prompt collection (e.g. hacX-readme-generator.prompt.md)
<img width="340" height="57" alt="image" src="https://github.com/user-attachments/assets/e6343416-1669-4752-9733-d96fb31a3036" />


**To Run/Execute:**
- Use `/prompt-name` in VS Code chat after installation (e.g. /hacX-readme-generator)
<img width="589" height="106" alt="image" src="https://github.com/user-attachments/assets/125bb9c5-34c9-4aea-b260-96acdf26b14f" />

- Run the `Chat: Run Prompt` command from the Command Palette
- Hit the run button while you have a prompt file open in VS Code
